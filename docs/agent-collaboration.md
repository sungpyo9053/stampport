# Stampport Agent Collaboration

## Core Factory Philosophy

Stampport must not be developed as a simple coding pipeline.

The planner agent and designer agent are the core of the factory.

The planner agent continuously proposes new service loops, rewards, badges, titles, quests, and progression systems.

The designer agent challenges whether those rewards are visually desirable, collectible, and shareable.

The developer agents implement only after planner/designer alignment.

## Planner Agent

The planner agent is not a simple requirement writer.

The planner agent must propose service upgrades every cycle.

Every planning output must include:

1. New service feature or reward proposal
2. User motivation
3. Connection to stamp, EXP, badge, title, quest, passport, or share card
4. Smallest shippable version
5. Designer challenge request

Example:
- Local Visa Badge
- Taste Title Evolution
- Passport Empty Slots
- Monthly Taste Report
- Rare Stamp Collection
- Same Stamp Holder Room

## Designer Agent

The designer agent is not a UI decorator.

The designer agent turns features into objects users want to own, collect, and show off.

Every design output must challenge:

1. Is this badge worth earning?
2. Is this title worth displaying?
3. Is this stamp worth collecting?
4. Is this card worth posting on Instagram Story?
5. Does the empty slot make users want the next visit?
6. Does it feel like passport/RPG/taste collection?

## Required Planner-Designer Ping-Pong

Every cycle must include:

1. Planner proposes 3 service upgrade candidates.
2. Designer critiques each candidate.
3. Planner revises the best candidate.
4. PM scopes the smallest shippable version.
5. Developer implements.
6. QA verifies function and emotional reward.

## QA Must Check

QA must check not only whether the feature works.

QA must check whether the feature supports:

- collection desire
- show-off desire
- progression desire
- scarcity desire
- next-visit motivation

A feature passes only if it works functionally and strengthens Stampport's local taste RPG loop.
