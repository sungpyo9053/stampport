# Stampport Design System

## Visual Identity

Stampport should feel like a local passport and RPG collection book.

Keywords:
- passport
- stamp
- badge
- title
- quest
- local exploration
- warm analog paper
- premium but playful

## Color Direction

Primary:
- Deep Green
- Cream
- Gold

Secondary:
- Burgundy
- Navy
- Warm Brown

## UI Components

Required components:
- PassportCover
- StampCard
- StampForm
- StampResult
- BadgeCard
- TitleCard
- QuestCard
- PassportSummary
- ShareCard

## Button Language

Use:
- 스탬프 찍기
- 내 여권 보기
- 칭호 확인하기
- 공유 카드 만들기
- 이번 주 퀘스트 보기

Avoid:
- 등록하기
- 제출하기
- 리뷰 쓰기

## Layout Rules

- Mobile-first.
- 390px width must be safe.
- Cards should feel collectible.
- Stamp result must feel rewarding.
- Passport page should look like a collection book.
- Badges should look earned, not like plain labels.

## Visual Rules

Do not make it look like:
- Naver Map
- a generic review app
- a plain admin dashboard
- a todo app

Make it feel like:
- local exploration
- passport stamping
- RPG progression
- taste collection
- SNS-shareable achievement
